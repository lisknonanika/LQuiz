What's LQuiz
LQuiz is a PoC built with LiskSDK.
Account initialization plays an important role in Lisk.
However, I must create a transfer transaction to initialize it.
Buy LSK, transfer LSK to my account, and transfer LSK again to initialize.ã€€(To be honest, this is troublesome lol)
When the conditions were met, I wanted to initialize my account and get an LSK.

So I thought this LQuiz.

The condition for initializing an account and obtaining an LSK is to answer the quiz correctly.
There is no fee for answer.

Next LQuiz
I think that LQuiz can be used for job requests by extending it a little more.
After confirming the completion of the work, you can give a pre-prepared reward by giving the "answer".
The fact that the reward is pre-prepared will give the work contractor peace of mind.

However, it is possible to enter the answer before the contractor, which could lead to fraud.
Therefore, there is a need for a system that incorporates some evaluation method and gives punishment.
How To
- Login Page -
  • Login Button: Enter your passphrase and login
  • Guest Login Button: Login without entering a passphrase (cannot create or answer questions.)
  • Create Account: Create an account that can be used with Lisk and LQuiz.

- Open Question Page -
  • Displays a list of currently available questions. (Questions registered by the login user is not displayed.)

- Close Question Page -
  • Displays a list of questions that have reached the number of reward recipients. (Questions answered by the login user is not displayed.)

- Create Question Page -
  • Create a new question. (Registration requires (reward * number of people receiving the reward) + fee.)

- My Question Page -
  • Questions registered by the login user is displayed.

- My Answered Page -
  • Questions Answerd by the login user is displayed.

- Change Account -
  • Switch login user.

- Logout -
  • Logout of Quiz.
